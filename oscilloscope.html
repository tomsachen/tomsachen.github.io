use this tool to modulate the parameters of a 3D torus knot with an audible spectrum sound wave.

eventually, this will function as an external software oscilloscope for physical instruments, and will include support for polyphonic voices. it will also act more capably as a stand-alone visual FM synthesizer.

WASD = rotate // Enter = reset rotation // Space = pause modulation // M = play modulation signal

a = outer radius, b = inner radius, p = long. winding, q = merid. winding
